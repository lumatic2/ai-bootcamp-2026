1. 모바일 상품 상세의 4개 이미지 세로 나열을 가로 스냅 갤러리로 변경했다.
2. 모바일 상품 상세에 가격과 구매 버튼을 포함한 하단 고정 CTA를 추가했다.
3. 모바일·태블릿 상품 목록은 필터 사이드바 대신 Sheet를 사용하도록 유지했다.
4. 데스크톱 5열, 태블릿 3열, 모바일 2열의 카드 폭을 확인했다.
