## 성공 모습
- 초기 문제 정의부터 제품 패키징까지 주요 결정마다 근거, 반증 조건, 담당자와 결과가 추적된다.
- 작동하는 최소 제품과 실제 또는 명시적으로 한계가 표시된 측정 데이터가 연결된다.
- 팀이 데이터에서 배운 내용과 다음 가설을 설명하고 같은 과정을 다시 실행할 수 있다.

- **학습 속도**: 마지막 날 회고 → 각 실험 직후 데이터 기반 학습과 다음 행동 결정.
