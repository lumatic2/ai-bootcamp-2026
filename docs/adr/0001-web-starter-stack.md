# ADR 0001 — 범용 웹 스타터 스택

- 상태: Accepted
- 날짜: 2026-07-14

## 맥락
팀이 3일 동안 아이디어를 정한 뒤 즉시 웹 제품을 만들 수 있어야 하지만, 제품 종속 구조를 미리 고정하면 안 된다.

## 결정
Next.js App Router, TypeScript, pnpm, Tailwind CSS v4, shadcn/ui를 사용한다. 백엔드, 인증, DB, AI API와 배포는 제외한다.

## 근거
- Next.js는 화면과 얇은 서버 기능을 같은 프로젝트에서 확장할 수 있다.
- Tailwind v4와 shadcn/ui는 토큰 기반의 빠른 UI 구현과 소유 가능한 컴포넌트를 제공한다.
- 팀 합류 전에 제품 도메인과 무관한 공통 환경만 준비할 수 있다.

## 결과
- 팀은 문제 선정 후 제품 surface에 집중할 수 있다.
- 실제 제품 요구가 SPA나 모바일에 더 적합하면 이 결정을 새 ADR로 교체한다.
