# HORIZON — 3일 Build–Measure–Learn 루프

> 생성: 2026-07-14 · ROADMAP marker: `harness:goal id="bootcamp-3day-loop"` · 상태: active

## 목표
- 팀이 문제 정의, 제품 구현, 측정, 학습, 발표를 하나의 증거 체인으로 연결한다.

## 왜 지금
- 부트캠프 시작 전 빈 레포만 있어 팀이 합류하면 환경 설정과 진행 방식 합의에 시간을 쓰게 된다.
- 제품 아이디어는 팀이 함께 결정해야 하므로, 아이디어와 무관한 실험 절차와 범용 웹 기반만 먼저 준비한다.

## 담을 milestone
- [ ] **B1** 팀 논의 전 실험·구현 기반 준비 — playbook, 디자인 계약, 웹 스타터를 통합 검증한다. → `docs/plans/2026-07-14-bootcamp-foundation.md`
- [ ] **B2** 팀 문제·솔루션 선정과 최소 실험 실행 — 문제 증거부터 학습까지 실제 데이터로 연결한다.
- [ ] **B3** 제품 패키징과 발표 evidence 구성 — 제품과 과정 증거를 평가자 관점에서 재현한다.

## 닫는 기준
- B1~B3의 DoD가 모두 충족되고, 발표에서 문제 → 가설 → 제품 → 데이터 → 학습 → 다음 가설을 역추적할 수 있다.

## Objective 임팩트
- close 시 결정 추적성, 검증 가능성, 학습 속도 세 축을 증거 수와 누락 항목으로 재측정한다.
